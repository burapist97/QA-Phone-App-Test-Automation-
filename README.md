# QA-Phone-App-Test-Automation-
Create own checklist, review steps, catch issues' screenshots and catch logs.

Türkçe arayüzüyle, telefon uygulamalarınız için test caseleri yaratın, uygulama sizin yerinize log toplasın screenshot alsın.

Kullanmadan önce;
Telefonun USB Hata ayıklamasının açık olduğuna ve telefona bağlanabildiğine emin olun
Telefonun bağlanması için adb dosyaları klasörün içerisinde mevcuttur uygulama sizin için bunu yükler ve denetler
Bilgisayarınızda Pyhton yüklü olması yeterlidir.
Basit ve kolay arayüzü ile hiç yazılım bilemeseniz bile istediğiniz uygulamanın test otomasyonunu yapabilirsiniz.


Çalışma mantığı;
Test başlat dediğinizde otomatize etmek istediğiniz test caselerini bir kere başarılı yapmanız yeterlidir
Testinizi yapın "Enter" tuşuna basın isim verin sonra ki adımı yaratmaya başlayın ve tekrar aynı adımları yaparak test caselerinizi yaratın
Uygulama cihazda tıkladığınız kordinatları kayıt ederek başarılı ekranlarda ki yazıları okur, test başladığında belirlenen adıma ki kordinatta bulunan adımlara tıklar sonuçları için sayfaları karşılaştırır.
Yarattığınız testleri testlerim sayfasında görüntüleyebilir aynı uygulamaya sahip kişilerle paylaşabilirsiniz
Test sonuçlarınız database üzerinde saklanır bu arayüzden test sonuçlarına tıklayarak hataların ekran görüntülerini ve loglarına ulaşabilirsiniz.


Not;
Uygulama hala geliştirme sürecindedir, bu süreç içerisinde uygulama hata yapabilir.
Kaydırma özelliği henüz güncellenmemiştir, gelecek sürümde kaydırma özelliğide eklenecektir.


Artık telefon uygulamaları için kimsenin yazılım bilmesine gerek kalmayacak.
